# Changelog

All notable changes to `cartify` will be documented in this file.

## [1.0.0] - 2025-01-18

### Added

- Initial release
- Shopping cart management with session storage
- Multiple cart instances support
- User persistence (store/restore/merge)
- Automatic calculations (subtotal, tax, total)
- Database migrations for persistent storage
- Helper functions (cart(), format_price(), generate_order_number())
- Full Laravel 12 support
- Comprehensive documentation
- MIT License

### Features

- Add/update/remove items
- Search cart items
- Count total items
- Check if cart is empty
- Instance-based carts (cart, wishlist, etc.)
- Tax calculations
- Price formatting
- Order number generation
